Apaptation vs Prediction
========================

–ú–æ–∂–Ω–æ –µ—â–µ –ø—Ä–æ–≤–µ—Å—Ç–∏ —Ç–∞–∫—É—é –∞–Ω–∞–ª–æ–≥–∏—é: Code Review - —ç—Ç–æ —Ñ–æ—Ä–º–∞ –∏–Ω—Å–ø–µ–∫—Ü–∏–∏ –∏ –∞–¥–∞–ø—Ç–∞—Ü–∏–∏, —Ç.–µ. —ç–º–ø–∏—Ä–∏—á–µ—Å–∫–∏–π (–æ–ø—ã—Ç–Ω—ã–π) —Å–ø–æ—Å–æ–± –æ–±—Ä–∞–±–æ—Ç–∫–∏ –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ—Å—Ç–∏. –≠—Ç–æ —Å—Ä–æ–¥–Ω–∏ —Ö–æ–∂–¥–µ–Ω–∏—é –Ω–∞ –æ—â—É–ø. –ö—É–¥–∞-—Ç–æ –¥–æ–±—Ä–∞—Ç—å—Å—è —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º –º–æ–∂–Ω–æ, –Ω–æ –ø—Ä–æ–∏—Å–ø–µ–∫—Ç–∏—Ä–æ–≤–∞—Ç—å —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º –º–æ–∂–Ω–æ —Ç–æ–ª—å–∫–æ —Ç–æ, —á—Ç–æ –º–æ–∂–Ω–æ –ø–æ—Ç—Ä–æ–≥–∞—Ç—å, —Ç.–µ. –∫–æ–≥–¥–∞ "—É–∂–µ –ø—Ä–∏—à–ª–∏".

–ü—Ä–æ—Ü–µ—Å—Å –º–æ–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–µ–π, –µ—Å–ª–∏ —Ä–∞—Å—à–∏—Ä–∏—Ç—å –≥–æ—Ä–∏–∑–æ–Ω—Ç "–≤–∏–¥–µ–Ω–∏—è", —Ç.–µ. –≤–Ω–µ—Å—Ç–∏ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—É—é –¥–æ–ª—é Prediction-–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏, –∏ –ø–æ–ª—É—á–µ–Ω–∏–µ –æ—Å–Ω–æ–≤–Ω–æ–≥–æ –∏–Ω–∫—Ä–µ–º–µ–Ω—Ç–∞ –∑–Ω–∞–Ω–∏–π –ø–µ—Ä–µ–Ω–µ—Å—Ç–∏ —Å Code Review –Ω–∞ Design Review.
